## 🔧 トラブルシューティング

### よくある問題と解決方法

#### 1. ページが正しく表示されない

**症状**: レイアウトが崩れている、要素が表示されない

**解決方法**:
1. ブラウザのキャッシュをクリア
   - Chrome: `Ctrl/Cmd + Shift + Delete`
   - Firefox: `Ctrl/Cmd + Shift + Delete`
   - Safari: `Cmd + Option + E`
2. ページを強制リロード
   - `Ctrl/Cmd + Shift + R`
3. ブラウザを最新版に更新

#### 2. ダークモードが切り替わらない

**症状**: テーマトグルボタンをクリックしても変化がない

**解決方法**:
1. ブラウザのJavaScriptが有効か確認
2. ローカルストレージをクリア
3. ページをリロード

#### 3. 言語が切り替わらない

**症状**: 言語セレクターで選択しても表示が変わらない

**解決方法**:
1. ページをリロード
2. ブラウザのCookieを確認
3. 別のブラウザで試す

#### 4. キーボードショートカットが動作しない

**症状**: ショートカットキーを押しても反応がない

**解決方法**:
1. フォーカスが適切な要素にあるか確認
2. ブラウザの拡張機能が干渉していないか確認
3. OSのキーボード設定を確認

#### 5. スクリーンリーダーで読み上げられない

**症状**: スクリーンリーダーが要素を認識しない

**解決方法**:
1. スクリーンリーダーを再起動
2. ブラウザのアクセシビリティ設定を確認
3. 最新のスクリーンリーダーバージョンを使用

## 10. サイドバーとナビゲーション

### 10.1 サイドバーの開閉

サイドバーは画面左側に配置され、チャット履歴や設定にアクセスできます。

**開閉方法**:
- 左上のメニューアイコン（☰）をクリック
- キーボードショートカット: `Ctrl+B`

### 10.2 チャット履歴の検索

サイドバー上部の検索ボックスで、過去のチャット履歴を検索できます。

**検索方法**:
1. サイドバーを開く
2. 検索ボックスに検索キーワードを入力
3. リアルタイムで検索結果が表示されます

**検索対象**:
- チャットのタイトル
- メッセージの内容
- 日付

**キーボードショートカット**: `Ctrl+F`でサイドバーの検索ボックスにフォーカス

### 10.3 チャット履歴の整理

チャット履歴は自動的に以下のセクションに分類されます：

- **最近**: 過去24時間以内のチャット
- **今週**: 過去7日間のチャット
- **過去**: 7日以上前のチャット

各セクションは展開・折りたたみが可能です。

### 10.4 チャットの削除

不要なチャット履歴を削除できます。

**個別削除方法**:
1. サイドバーでチャット履歴にマウスをホバー
2. 表示される削除アイコン（❌）をクリック
3. 確認ダイアログで「OK」をクリック

**全履歴削除方法**:
1. サイドバー下部の「🗑️ 全履歴削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

**削除確認ダイアログ**:
- 日本語: 「このチャット履歴を削除しますか？」
- 英語: "Are you sure you want to delete this chat history?"
- 全削除（日本語）: 「全てのチャット履歴を削除しますか？」
- 全削除（英語）: "Are you sure you want to delete all chat history?"

**注意**: 
- 削除したチャットは復元できません
- 削除は即座にサーバーに反映されます
- 削除後、UIは自動的に更新されます

**実装詳細（開発者向け）**:
- API: `DELETE /api/chat/history?sessionId={id}`
- 全削除API: `DELETE /api/chat/history?all=true`
- レスポンス: `{ success: true, message: "...", sessionId: "..." }`
- エラーハンドリング: トースト通知で表示

### 10.5 設定パネル

画面右上の歯車アイコンから設定パネルにアクセスできます。

**設定項目**:

#### 一般設定
- 言語選択（日本語/English）
- デフォルトモデルの設定
- 自動保存の有効/無効

#### 外観設定
- テーマ選択（ライト/ダーク/システム）
- フォントサイズ調整
- サイドバーの表示/非表示

#### 通知設定
- デスクトップ通知の有効/無効
- サウンド通知の有効/無効
- 音量調整

#### 高度な設定
- チャット履歴の自動削除
- データのエクスポート/インポート
- キャッシュのクリア

**キーボードショートカット**: `Ctrl+,`で設定パネルを開く

### 10.6 パフォーマンス

大量のチャット履歴（1000件以上）でもスムーズに動作するよう、仮想スクロール技術を採用しています。

**特徴**:
- 高速なスクロール
- メモリ効率の最適化
- 検索結果の即座表示



## 11. モデル選択

### 11.1 モデル選択の開き方

AIモデルを変更するには、ヘッダーのモデル名ボタンをクリックします。

**開き方**:
- ヘッダーの「✨ モデル名」ボタンをクリック
- キーボードショートカット: `Ctrl+K`

### 11.2 モデルの選び方

モデル選択画面では、利用可能なAIモデルがカード形式で表示されます。

**モデルカードの見方**:
- **プロバイダー名**: Amazon、Anthropic、Meta、Cohere、Mistral AI、AI21 Labs
- **モデル名**: 各モデルの正式名称
- **利用可能状態**: 
  - ✅ 利用可能
  - ❌ 利用不可
  - 🔒 サブスクリプション必要
- **性能指標**:
  - 📊 最大トークン数
  - ⚡ レイテンシ（低/中/高）
  - 💰 コスト（$/$$/$$$）

**選択方法**:
1. モデルカードをクリック
2. 確認ダイアログが表示される
3. 「変更する」をクリックして確定

### 11.3 フィルター機能

左側のフィルターパネルで、表示するモデルを絞り込めます。

**フィルター項目**:

#### プロバイダーフィルター

### 11.6 モデル変更の注意点

**変更時の動作**:

- チャット履歴は保持されます

**変更の確認**:

### 11.7 トラブルシューティング

**モデルが選択できない**:

**フィルターで何も表示されない**:
- フィルター条件を緩和してください
- 「クリア」ボタンですべてのフィルターを解除できます

**モデル変更が反映されない**:
- ページをリロードしてください
- ブラウザのキャッシュをクリアしてください

## 12. PWA機能とオフライン対応

### 12.1 ホーム画面への追加

**インストールプロンプト**:
アプリケーションをホーム画面に追加できます。

**表示条件**:
- PWA対応ブラウザを使用している
- 初回訪問から一定時間経過後
- 過去に「後で」を選択した場合は7日後に再表示

**インストール手順**:

**iOS (Safari)**:
1. 共有ボタン（□↑）をタップ
2. 「ホーム画面に追加」を選択
3. 名前を確認して「追加」をタップ

**Android (Chrome)**:
1. インストールプロンプトの「インストール」をタップ
2. または、メニュー（⋮）→「ホーム画面に追加」

**デスクトップ (Chrome/Edge)**:
1. アドレスバーのインストールアイコンをクリック
2. または、インストールプロンプトの「インストール」をクリック

### 12.2 オフライン機能

**オフライン時の動作**:
- 既存のチャット履歴は閲覧可能
- 新しいメッセージは送信キューに保存
- オンライン復旧時に自動送信

**メッセージキュー**:
- オフライン時に送信したメッセージは自動的にキューに追加
- キューステータスがヘッダーに表示
- オンライン復旧時に自動的に送信を試行

**キューステータス表示**:
```
📤 送信待ち: 3件
```

**手動送信**:
キューステータスをクリックすると、手動で送信を試行できます。

### 12.3 オフラインページ

**表示条件**:
- ネットワーク接続が完全に切断されている
- 新しいページに移動しようとした場合

**オフラインページの機能**:
- ネットワーク接続状態の表示
- オンライン復旧の自動検出
- 「再試行」ボタン
- トラブルシューティングヒント

**復旧時の動作**:
オンラインに復旧すると、自動的に元のページに戻ります。

### 12.4 キャッシュ管理

**キャッシュされるコンテンツ**:
- アプリケーションのコアファイル
- 静的アセット（画像、フォント）
- 最近のチャット履歴

**キャッシュのクリア**:
ブラウザの設定から「サイトデータを削除」を実行してください。

### 12.5 バックグラウンド同期

**自動同期**:
- オンライン復旧時に自動的にメッセージを送信
- バックグラウンドで実行されるため、アプリを開いていなくても同期

**同期の確認**:
- 送信成功時にトースト通知が表示
- チャット履歴に送信済みメッセージが反映

### 12.6 PWAのメリット

**ユーザーメリット**:
- ホーム画面から直接起動
- ネイティブアプリのような体験
- オフラインでも基本機能が利用可能
- 高速な起動とページ遷移

**技術的メリット**:
- プッシュ通知対応（将来実装予定）
- バックグラウンド同期
- 効率的なキャッシュ管理
- 低帯域幅環境でも快適
