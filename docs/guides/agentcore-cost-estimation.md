## 概要

### 試算の目的

Amazon Bedrock AgentCoreの9つの機能コンポーネントをステージング環境と本番環境で稼働させた場合の月額コストを試算します。

### 試算の前提

- **リージョン**: ap-northeast-1（東京）
- **稼働時間**: 24時間365日（本番）、平日8時間（ステージング）
- **リクエスト数**: 想定ワークロードに基づく
- **データ量**: 想定データ量に基づく
- **価格**: 2026年1月時点の東京リージョン価格

### 価格参照元

- AWS公式価格ページ（2026年1月時点）
- AWS料金計算ツール
- 実際の価格は変動する可能性があります

## ステージング環境コスト試算

### 1. Lambda（9機能）

#### 料金体系（東京リージョン）
- **リクエスト料金**: $0.0000002 per request
- **実行時間料金**: $0.0000166667 per GB-second

#### 計算
```
リクエスト数: 10,000回/月
実行時間: 10,000回 × 3秒 = 30,000秒
メモリ: 1024MB = 1GB
GB-秒: 30,000秒 × 1GB = 30,000 GB-秒

リクエスト料金: 10,000 × $0.0000002 = $0.002
実行時間料金: 30,000 × $0.0000166667 = $0.50
無料枠: -$0.50（初月のみ）

月額コスト: $0.50
```

**Reserved Concurrency追加コスト**:
```
9機能 × 5並行 = 45並行
Reserved Concurrency料金: なし（実行時間のみ課金）
```

### 2. DynamoDB（3テーブル）

#### 料金体系（東京リージョン）
- **書き込み**: $1.4269 per million write request units
- **読み取り**: $0.285 per million read request units
- **ストレージ**: $0.285 per GB-month

#### 計算
```
書き込み: 10,000回/月 = 0.01M
```

## コスト最適化推奨事項

### 1. FSx for ONTAPの最適化

**現状の課題**:
- ステージング: $449.54/月（全体の89%）
- 本番: $899.07/月（全体の20%）

**重要**: FSx for ONTAPはこのプロジェクトの中核機能であり、高度なデータ管理・権限制御・スナップショット機能を提供します。S3/EFSへの代替は不可。

**最適化案**:

#### オプション1: スループット最適化
```
ステージング: 128MB/s → 64MB/s
- ワークロード分析に基づく適切なスループット設定
- テスト環境では低スループットで十分な場合が多い
削減額: 64MB/s × $2.20 = $140.80/月（31%削減）

本番: 256MB/s → 192MB/s（ワークロード次第）
- ピーク時のスループット要件を分析
- 必要最小限のスループット設定
削減額: 64MB/s × $2.20 = $140.80/月（16%削減）
```

#### オプション2: ストレージ容量の最適化
```
ステージング: 1TB → 512GB
- テストデータのみで十分な場合
- 定期的なデータクリーンアップ
削減額: 512GB × $0.164 = $83.97/月（50%削減）

本番: データ重複排除・圧縮機能の活用
- FSx for ONTAPの重複排除機能で実効容量削減
- 圧縮率: 平均30-50%
削減額: 約$100-170/月（30-50%削減）
```

#### オプション3: スナップショット管理の最適化
```
- 不要なスナップショットの定期削除
- スナップショット保持期間の最適化
- ステージング: 7日保持
- 本番: 30日保持
削減額: 約$20-50/月
```

#### オプション4: FSx for ONTAP階層化の活用
```
- Capacity Pool Tier（S3）への自動階層化
- アクセス頻度の低いデータを自動的にS3に移動
- S3ストレージコスト: $0.025/GB（FSxの15%）
削減額: 約$50-100/月（階層化データ量による）
```

## 推奨事項

### 短期的な対応（即座に実施可能）

1. **ステージング環境のFSxスループット最適化**
   - スループットを128MB/s → 64MB/sに削減
   - テスト環境では低スループットで十分
   - 削減額: $140.80/月

2. **CloudWatch Logsの保持期間短縮**
   - 30日 → 14日に変更
   - 削減額: ステージング$3.59/月、本番$39.27/月

3. **不要なCloudWatch Alarmsの削除**
   - 使用していないアラームを削除
   - 削減額: 約$1-2/月

4. **FSx for ONTAPスナップショット管理**
   - 不要なスナップショットの定期削除
   - ステージング: 7日保持、本番: 30日保持
   - 削減額: 約$20-50/月

### 中期的な対応（1-3ヶ月）

1. **DynamoDB Reserved Capacityの購入**
   - 1年契約で30%割引
   - 削減額: $864.12/月
