# PWAスクリーンショット

このディレクトリには、PWA（Progressive Web App）用のスクリーンショットを配置します。

## 必要なスクリーンショット

### デスクトップ版
- desktop-1.png (1920x1080)
  - チャットインターフェースのメイン画面
  - サイドバー、メッセージリスト、入力エリアを含む

### モバイル版
- mobile-1.png (750x1334)
  - モバイルビューのチャット画面
  - レスポンシブデザインを示す

## スクリーンショット撮影方法

### デスクトップ版
1. ブラウザを1920x1080に設定
2. アプリケーションを開く
3. サンプルチャットを表示
4. スクリーンショットを撮影

### モバイル版
1. Chrome DevToolsのデバイスモードを使用
2. iPhone 8 Plus (750x1334) を選択
3. アプリケーションを開く
4. サンプルチャットを表示
5. スクリーンショットを撮影

## 注意事項

- 個人情報や機密情報を含めない
- サンプルデータを使用する
- 高品質な画像（PNG形式）を使用
- ライトモードとダークモードの両方を考慮
