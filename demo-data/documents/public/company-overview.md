# 会社概要 - NetApp Japan Tech Team

## 事業概要

NetApp Japan Tech Teamは、クラウドストレージとデータ管理ソリューションを提供する技術チームです。
Amazon FSx for NetApp ONTAPを活用したエンタープライズグレードのデータ管理基盤を構築しています。

## 主要サービス

- クラウドストレージ最適化コンサルティング
- FSx for ONTAP導入支援
- データライフサイクル管理ソリューション
- ハイブリッドクラウドアーキテクチャ設計

## 技術スタック

- Amazon FSx for NetApp ONTAP
- Amazon Bedrock (RAG)
- AWS CDK v2
- Next.js / React
